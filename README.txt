Ficheiros:

file.py - Inclui os métodos BZIP2, DEFLATE e LZMA
rle.py - Inclui o RLE
sais.py - Inclui as funções necessárias para criar suffix array
metodo.py - Método Final criado pelos estudantes


Nos topo dos ficheiros, estão definidas as seguintes variáveis:
source_file -> Nome do ficheiro que pretendemos comprimir/descomprimir
extension -> Extensão final do ficheiro comprimido
destination_file -> Nome do ficheiro comprimido


Para testar os métodos BZIP2,DEFLATE e LZMA basta fazer run do ficheiro file.py, e escolher o método que pretende.
Para testar o RLE, tem de dar run do ficheiro rle.py.
Para testar o método criado por nós, tem de dar run do ficheiro metodo.py.

Extra: As imagens têm que estar na mesma pasta que os ficheiros python.
